# Flavorly App - Professional Installation

## 🍳 Welcome to Flavorly!

Transform your ingredients into extraordinary meals with AI-powered recipes.

## 📱 Installation Instructions

### For Windows Users:
1. Download `Flavorly-Full-App.zip`
2. Extract the zip file
3. Double-click `install.bat`
4. Follow the installation prompts
5. Launch from desktop shortcut or Start Menu

### For macOS Users:
1. Download `Flavorly-Full-App.zip`
2. Extract the zip file
3. Open Terminal and run: `./install.sh`
4. Launch from desktop shortcut

### For Linux Users:
1. Download `Flavorly-Full-App.zip`
2. Extract the zip file
3. Run: `./install.sh`
4. Launch from applications menu

## 🛠️ Troubleshooting

**App won't open?**
- Make sure you have a modern web browser
- Try opening `app.html` directly in your browser

**Installation failed?**
- Make sure you have administrator permissions
- Try running the installer as administrator

**Can't find the app?**
- Check your desktop for the shortcut
- Look in your Start Menu/Applications folder

## 🗑️ Uninstall

To remove Flavorly:
- **Windows:** Run `uninstall.bat` from the app folder
- **macOS/Linux:** Run `uninstall.sh` from the app folder
